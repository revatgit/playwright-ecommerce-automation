# Playwright Ecommerce Automation Framework

This project demonstrates an end-to-end UI automation framework built using Playwright and JavaScript.

## Features

- Page Object Model (POM)
- Data-driven testing
- End-to-end ecommerce flow automation
- Modular test structure

## Test Scenarios

1. User login
2. Product search
3. Add product to cart
4. Verify cart contents

## Tools Used

- Playwright
- JavaScript
- Node.js
- GitHub

## Run Tests

npm install
npx playwright install
npx playwright test
